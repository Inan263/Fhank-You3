Option 2: Using CSS Grid

CSS Grid is another powerful tool for creating side-by-side layouts. Here’s how you can do it:

HTML Structure

The HTML structure remains the same as in the flexbox example.

CSS Styling with CSS Grid

css
.product-container {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
    gap: 1rem; /* Space between products */

.product {
    background: white;
    border: 1px solid #ddd;
    border-radius: 8px;
    padding: 1rem;

Responsive Design Considerations

  • Flexbox: The flex-wrap: wrap; property ensures that products will wrap to the next line if the screen size is too small.
  • CSS Grid: The auto-fill and minmax() functions ensure that the grid adjusts automatically based on the screen size, making it responsive.

Choosing Between Flexbox and Grid

  • Flexbox is simpler for straightforward row/column layouts and works well when you want items to wrap naturally.
  • CSS Grid offers more control over complex layouts and allows you to define rows and columns precisely.

Both methods will display your products side by side effectively, so you can choose based on your specific needs or layout complexity.

3. Add JavaScript to Toggle Product Details

To enable the functionality of showing and hiding the product details when the toggle button is clicked, you need to add JavaScript.

JavaScript Code

html

<script>
    document.addEventListener('DOMContentLoaded', () => {
        // Select all toggle buttons
        const toggleButtons = document.querySelectorAll('.toggle-details');

        toggleButtons.forEach(button => {
            button.addEventListener('click', () => {
                // Find the corresponding product-details div
                const product = button.parentElement;
                const details = product.querySelector('.product-details');

                // Toggle the 'show' class
                details.classList.toggle('show');

                // Update the button text
                if (details.classList.contains('show')) {
                    button.textContent = 'Hide Details';
                } else {
                    button.textContent = 'Show Details';
                }
            });
        });
    });
</script>

Explanation:

  • Event Listener: Waits until the DOM is fully loaded before running the script.
  • Selecting Buttons: It selects all elements with the class .toggle-details.
  • Adding Click Event: For each toggle button, it adds a click event listener.
  • Toggling Details: When clicked, it finds the .product-details div within the same .product container and toggles the show class to display or hide it.
  • Updating Button Text: Changes the button text to "Hide Details" when details are shown and back to "Show Details" when hidden.
